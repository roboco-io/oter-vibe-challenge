# 제출 안내

## 1. 템플릿 복제

과제용 템플릿 저장소를 **"Use this template"** 버튼으로 **Private** 저장소로 복제합니다. (Owner는 본인 계정, Repository visibility는 Private)

- 템플릿에서 새 저장소 만들기: https://github.com/roboco-io/oter-vibe-challenge-template/generate
- 템플릿 저장소: https://github.com/roboco-io/oter-vibe-challenge-template

## 2. 데일리 push (필수)

과제 기간 동안 **매일 커밋을 push**합니다. 진행 상황과 사고 과정이 커밋 이력에 드러나야 하며, 데일리 push를 지키지 않으면 감점됩니다.

## 3. 마감 시 제출 절차

1. 저장소 Settings → Collaborators에서 GitHub 계정 **`serithemage`** 를 Collaborator로 초대합니다.
2. 최종 결과물을 push합니다.

## 4. 예산 정산

- 지원자가 자비로 NHN Cloud 비용을 부담하며, 제출 완료 시 **5만원을 보전**합니다.
- 소형 인스턴스 사용, 과제 종료 후 자원 teardown을 권장합니다. 예산 범위 내에서 운영하는 것 자체가 평가 대상입니다.

## 5. 금지 사항

- **시크릿(API 키, 자격 증명, `.env` 등) 커밋 절대 금지**. 발견 시 불합격 처리됩니다. `.gitignore` 설정과 커밋 전 점검을 습관화하세요.
